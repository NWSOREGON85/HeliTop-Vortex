WHY HELITOP VORTEX?

HELITOP VORTEX IS A MID-FIDELITY LAGRANGIAN VORTEX-FILAMENT SIMULATOR THAT RUNS IN SECONDS TO MINUTES ON A LAPTOP, YET DELIVERS PHYSICALLY REALISTIC RESULTS FOR COMPLEX WAKE FLOWS. IT COMBINES ADAPTIVE FILAMENTS, STOCHASTIC DIFFUSION, AUTOMATIC RECONNECTION, AND IMAGE VORTICES WITH A CLEAN, MODERN GUI.

INSTALLATION

WINDOWS (RECOMMENDED)
1. DOWNLOAD HELITOP_VORTEX_V18.0.EXE FROM THE RELEASES PAGE
2. RUN THE EXE - NO INSTALLATION NEEDED

FROM SOURCE
GIT CLONE HTTPS://GITHUB.COM/NWSOREGON85/HELITOP-VORTEX.GIT
CD HELI-TOP-VORTEX
PIP INSTALL NUMPY MATPLOTLIB PILLOW VTK CUPY-CUDA12X   # OR CUPY-CUDA13X FOR YOUR GPU
PYTHON HELI_TOP_GUI.PY

QUICK START
1. LAUNCH THE PROGRAM.
2. SELECT A PRESET (E.G., CIRCULAR_PIPE OR HVAC_PIPE).
3. LEAVE SLIDERS AT DEFAULTS (THEY ARE ALREADY OPTIMIZED FROM OUR SWEEPS).
4. CLICK RUN.
5. WATCH THE LIVE 3D PREVIEW AND CONSOLE.
6. WHEN FINISHED, CHECK:
   REPORTS/ -> FULL PDF REPORT
   PLOTS/ -> EFFICIENCY CURVE + ROTATING GIFS
   VTK/ -> VELOCITY FIELDS FOR PARAVIEW

PRESETS AND RECOMMENDED SETTINGS
CIRCULAR_PIPE -> DUCTED FANS, MARINE THRUSTERS
RECOMMENDED: CORE SIZE = 0.10, SWIRL LAYERS = 2 (PEAK EFFICIENCY ~0.242)

HVAC_PIPE -> HVAC DUCTS, VENTILATION SYSTEMS
RECOMMENDED: CORE SIZE = 0.10, SWIRL LAYERS = 2-3 FOR OPTIMAL SHEAR AND PRESSURE DROP

MARINE_PROPELLER -> SHIP PROPELLERS, UNDERWATER VEHICLES

ROCKET_PLUME -> ROCKET EXHAUST ANALYSIS (USE DYNAMIC AMPLITUDE FOR OSCILLATION)

CALIBRATION PANEL - DETAILED SLIDER GUIDE
CORE SIZE (0.01-0.2): VORTEX CORE RADIUS. 0.10 IS OPTIMAL FOR EFFICIENCY/STABILITY BALANCE.
SWIRL LAYERS (1-8): NUMBER OF CONCENTRIC VORTICITY RINGS. 2 LAYERS GIVE HIGHEST EFFICIENCY AND LOWEST TORQUE.
AXIAL STRENGTH (0.5-2.0): MAIN FLOW SPEED IN PIPES. HIGHER VALUES INCREASE VELOCITY AND DELTA P.
FRICTION (0.01-0.1): WALL FRICTION FACTOR FOR PRESSURE-DROP CALCULATIONS.
RPM (100-3000): ROTATIONAL SPEED FOR PROPELLER PRESETS.
VISCOSITY X (0.5-2.0): DIFFUSION SCALING. DEFAULT 0.5 IS OPTIMAL.
CIRCULATION X / CORE X / THRUST SCALE X: FINE-TUNE OVERALL STRENGTH AND OUTPUT.
DYNAMIC AMPLITUDE (0.0-0.2): ADDS TIME-VARYING THRUST (ROCKET PLUMES).

USING THE CONFIG.JSON FILE
THE PROGRAM AUTOMATICALLY SAVES/LOADS CONFIG.JSON.
TO CUSTOMIZE:
1. RUN ONCE -> FILE IS CREATED.
2. CLOSE PROGRAM.
3. EDIT ANY VALUE (E.G., "NUM_SWIRL_LAYERS": 2, "CORE_BASE": 0.10).
4. SAVE AND RESTART - CHANGES LOAD AUTOMATICALLY.

HVAC EXAMPLE (PASTE INTO CONFIG.JSON):
{
  "PRESET": "HVAC_PIPE",
  "PIPE_RADIUS": 0.4,
  "NUM_SWIRL_LAYERS": 3,
  "AXIAL_STRENGTH": 1.5,
  "CORE_BASE": 0.10,
  "VISCOSITY_MULTIPLIER": 0.5
}
