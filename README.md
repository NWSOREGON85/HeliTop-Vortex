# HeliTop Vortex v8.0

**The best free vortex filament simulator for engineers**

Professional tool for marine propellers, aircraft wakes, wind turbines, and jet plumes.

## Features
- Beautiful GUI (no coding required)
- Marine propeller with automatic Kt/Kq, efficiency curves, and PDF reports
- Aircraft wake preset
- Instant VTK export for ParaView
- Reconnection movie with live highlights
- One-click PDF engineering report

## Quick Start (Windows)
1. Download the latest release
2. Double-click `run_heli_top.bat`
3. Choose preset and click RUN SIMULATION

## One-Click Executable
Double-click `build_exe.bat` to create `HeliTop_Vortex.exe` (no Python needed)
